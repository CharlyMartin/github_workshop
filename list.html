Collaborating with Github

If you're reading this, it means you've been able to successfully clone the repository from my GitHub account on your laptop. Congrats 👏

Now create a new branch using git to update this file locally: add your team name to the list below. If you don't have one, make one up 😁

Finally, push your branch on GitHub and create a Pull Request for me to accept it. Or not, I will check the quality of your code first 😉
